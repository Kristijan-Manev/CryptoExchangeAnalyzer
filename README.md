# CryptoExchangeAnalyzer

Веб апликација за анализа на криптовалути

---
Docker сликата е објавена и на Docker Hub:
https://hub.docker.com/r/manevkris/crypto-app

---

Апликацијата е деплојирана на Microsoft Azure Web Apps со Linux контејнер:

URL: https://crypto-exchange-analyzer-app-ejbkfgdqdbakdba2.germanywestcentral-01.azurewebsites.net

При пребарување (search) за криптовалутите etc, ens, bnb, апликацијата овозможува
податоците од овие криптовалути да се користат за техничка анализа и LSTM предикција.
Резултатите од анализите се визуелно прикажани преку веб интерфејс.
